Employee Appraisals Process and Evaluation Form

HR Evaluation of Employee Appraisals Odoo App

Employee Evaluation Form Backend Management

This app allows your HR / Evaluation Team to do the employee appraisal evaluation as shown the below screenshots.


Main Features:

  • Configuration of Appraisals/Evaluation Plans under configuration as shown.

  • The system has created two new groups called evaluation administrator and evaluation officer which you can find on user settings as shown.

  • The evaluation officer / HR officer can create and manage appraisals/evaluation for employees.

  • The system will auto-generate an evaluation form based on your configuration on the plan and allow the users to fill the form.

  • Top-Down, Bottom-Up, Self, and Final Appraisal Plans can be set up.

    • Self:- Employees can fill up the evaluation form.

    • Top-Down:- The employee-manager can fill up the evaluation form.

    • Bottom-Up:- Employees direct children can fill up the Evaluation Form. (That means this employee set as manager on the children employee).

    • Final :- Evaluation Officer/Manager or HR Officer can fill up the evaluation form.

  • The next appraisal date on the employee form also allow you to set an Appraisal plan on the employee form.

  • Allow your users to print the evaluation form.

  • Allow you to configure and manage evaluation question templates, competencies questions, object questions, and ratings as shown.

  • Please note that this app provides a feature that allows your users to fill out the form directly in the backend. You can configure the appraisal plan along with templates as shown in the below screenshots.

  • When you will start the appraisal the system will automatically create all the evaluation forms to fill by an employee, manager, children, and HR Officer.

  • The system can allow them to sign an evaluation form after completing and the sign will be stored on the form.

  • The system also allows them to give ratings on the evaluation form.

  • Also, allow generating appraisal in mass from a list of employees.

  • For more details please check the below screenshots and watch the video.
