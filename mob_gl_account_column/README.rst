General Ledger XLSX Account Name Column
========================================

This module extends the account_financial_report General Ledger XLSX export to add an 'Account Name' column at the end of the report. The column displays the account name for each move line, as shown in the report header.

Installation
------------
- Add this module to your Odoo addons path.
- Install it via the Odoo Apps interface.

Usage
-----
- Go to Accounting > Reporting > General Ledger.
- Print the General Ledger in XLSX format.
- The exported file will now include an 'Account Name' column at the end.

Dependencies
------------
- account_financial_report
