Advance Salary Request by Employees in Odoo

Advance Salary in Odoo

This module allows employees to request an advance salary.


Main Features:
  • Martin (HR Manager / Payroll Officer) can set the salary limit on Job Position.
  • Albert (employee) can log in the system and create advance salary requests.
  • David (Department Manager) can log in the system and approve department approval of advance salary requests.
  • Martin (HR Manager / Payroll Officer) can log in the system and approve HR approval of advance salary requests.
  • Grills Grave (Director) can log in the system and approve director approval of advance salary requests.
  • Mitchell John (Accountant) can log in the system and pay advance salary requests.
  • Payment Confirm to Employee by Mitchell John (Accounting Department). (Odoo Standard)
  • Payment Done to Employee by Mitchell John (Accounting Department). (Odoo Standard)
  • Martin (HR Manager / Payroll Officer) can see salary rules & salary structures. (Odoo Standard)
  • Martin (HR Manager / Payroll Officer) can compute payslip of employees. (Odoo Standard)
  • Print Advance Salary Request PDF Report.
  • For more details please check below screenshots and watch the video.

We have created Department Manager, Director groups under settings/users/groups to control the process.

Workflow: Draft->Confirmed->Approved by Department->Approved by HR->Approved by Director->Paid->Done.
